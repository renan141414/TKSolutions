
## Exercício 9: Exclusão de Documento no MongoDB

### Objetivo
Remover um documento da coleção "alunos" no MongoDB usando o nome como critério.

### Descrição
Solicitar ao usuário o nome do aluno a ser excluído e usar a função delete_one() para removê-lo.

### Entrada
Nome do aluno.

### Saída
Documento removido do banco de dados.
